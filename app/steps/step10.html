Attributes


@ is the syntax used for referring to attributes.
@attr - refers to the attribute 'attr'

You may use an attribute as condition for the predicate to specify that the element that you want to select uses that attribute.

Example:

elem[@attr] - all elements elem that contain the attribute @attr


Use [....], @ to select all the input elements that have the attribute id in the current context.
