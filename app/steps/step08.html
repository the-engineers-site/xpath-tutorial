Predicates: child elements


[...] is the syntax for a predicate. It is used to restrict a node-set to select only those nodes for which some condition is true

You may use a node as condition for the predicate to specify that the element that you want to select has that node as a child.

Example:

elem[child1[child1_1]] - all elements elem that contain at least one child element child1 which also contains a child element child1_1
